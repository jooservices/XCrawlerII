# User Guide

## Dashboard Overview

The XCrawlerII Dashboard provides a centralized view of the collected JAV metadata.

### Features

-   **Search**: Use the top search bar to find movies by title, code (e.g., `ABC-123`), actor, or tag.
-   **Filters**:
    -   **Date**: Filter by release date.
    -   **Genre**: Select specific tags/genres.
    -   **Actress**: Filter by specific actresses.
-   **Sorting**: Sort results by Release Date, Rating, or Added Date.

## Personalization

-   **Watchlist**: Click the "Bookmark" icon on any movie card to add it to your watchlist.
-   **Favorites**: Mark actors or movies as favorites for quick access.
-   **History**: Your browsing history is automatically saved (can be disabled in settings).

## Admin Features (If Authorized)

-   **Sync Status**: Monitor the status of active crawlers.
-   **Telemetry**: View real-time graphs of crawler performance and queue health.
